<h2> Getting Started</h2>

1. Install dependencies

   ```bash
   npm install
   ```

2. Test it

   ```bash
   npm test
   ```

3. Test it and check code-coverage

   ```bash
   npm run test:coverage
   ```

4. Start it up

   ```bash
   npm start
   ```
